# ELECTRIZ — India Election Visualization Dashboard for Media

An interactive Power BI dashboard analyzing India's 2024 Lok Sabha 
Election results with historical comparisons from 2014 and 2019.

## Dashboard Pages
- **Real-Time Results** — KPI cards, India map, party charts
- **Historical Trends** — 2014 vs 2019 vs 2024 comparison  
- **Turnout Analysis** — State and constituency voter participation
- **Demographics** — Gender, age group, geography breakdown
- **Party & Candidate Insights** — Deep dive into party performance

## Tools Used
- Microsoft Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query Editor
- Star Schema Data Modeling

## Repository Structure
- Power BI .pbix file 
- /datasets — All 8 CSV and Excel source files
- /screenshots — Dashboard page screenshots


## Key Findings
- BJP won 240 seats (below 272 majority — needed coalition)
- INC best performance since 2014 with 99 seats
- National average turnout: 67.73%
- Historic margin: Shankar Lalwani won Indore by 11.75 lakh votes
